# Case #0626 — Roudah Birthday Website

This is a GitHub Pages-ready website.

## Files to upload

Upload these files to your GitHub repository:

- `index.html`
- `style.css`
- `script.js`

## Publish with GitHub Pages

1. Open your repository.
2. Go to **Settings**.
3. Click **Pages** on the left.
4. Under **Build and deployment**, choose:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/ (root)`
5. Click **Save**.
6. Wait 1–2 minutes.
7. Your link will look like:
   `https://yourusername.github.io/roudah-birthday/`

## Notes

- This version does not include photos.
- It includes tulip animations, a confidential case-file theme, clickable gift rewards, and a Netherlands tulip fields voucher.


## Adding the real gift card codes

Open `script.js` and find this section:

```js
code: "Sent separately for your safety ❤️"
code: "PASTE-DISNEY-PLUS-CODE-OR-LOGIN-HERE"
code: "SENT-SEPARATELY-IN-BEANZ-APP"
```

Replace the placeholder text inside the quotation marks with the real code or message.

Examples:

```js
code: "X1Y2-Z3A4-B5C6-D7E8"
code: "Email: example@email.com / Password: example"
code: "Sent to you in Beanz app ❤️"
```

Do this before sending the website link.


## Safer gift delivery

This version intentionally does not place the real Apple gift-card code, Disney+ phone/login details, or Beanz gift inside the public GitHub Pages site.

Send those privately through WhatsApp, iMessage, or the relevant app after Roudah opens the site.
